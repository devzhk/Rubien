Rubien ‚Äî Obsidian Web Clipper same-origin reading assets
========================================================

Files in this directory:
- ClipperDefuddle.js     ‚Äî webpack output (Defuddle full + RubienDefuddleExtract)
- ClipperReader.css      ‚Äî copied from obsidian-clipper/dist/reader.css
- ClipperHighlighter.css ‚Äî copied from obsidian-clipper/dist/highlighter.css
- Readability.js         ‚Äî Mozilla Readability (Apache-2.0), fallback extractor

Important (diffs from obsidian-clipper upstream):
- Rubien does NOT load dist/reader-script.js, and does NOT invoke reader.ts `Reader.apply()`.
- If `siteName`/`journal` equals `url`, only one is shown in the header.
- Debugging the online-read pipeline: open Console.app ‚Üí filter subsystem `Rubien`, category `OnlineReadable`. The injection script posts phase names via `RubienClipperDebug` (e.g. `rubien_defuddle_extract_start`).
- `readerResult` is the WKScriptMessageHandler name. `ClipperDefuddle.js` `RubienDefuddleExtract` both `postMessage`s and returns JSON from `evaluateJavaScript` (dual channel). `ReaderExtractionManager` receives them and removes the handler on `deinit`.
- `ReaderExtractionManager` (Sources/Rubien/ReaderExtraction) handles Defuddle ‚Üí Readability, 1.5s delayed retry once, and a Safari-like `customUserAgent`.

To update Defuddle or the reader stylesheet:
  cd obsidian-clipper/
  npm install
  npm run build:rubien-defuddle

Then copy `dist/reader.css` / `dist/highlighter.css` over the files in this directory (if the styles changed).
